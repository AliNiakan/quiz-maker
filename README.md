<h1>Installation Guide: </h1>  <br/>
1.Build packages and project<br/>

```
npm init
``` 

<br/> 
2.Put API Url in config.json File (located in ./src/config.json)  <br/> <br/>
3.Start the project<br/>

```
npm start
```
